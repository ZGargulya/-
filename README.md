# Анализ транзакций сети розничных магазинов
## Курсовая работа по модулю обучения основам Excel
____
**Цель работы** - выявить операционные аномалии, оценить влияние маркетинговой акции на прибыль и средний чек,
визуализировать ключевые показатели бизнеса, собрать результаты технических заданий в единую структуру

**Легенда:** На протяжении нескольких месяцев, работая аналитиком в крупном торговом ретейлере, анализировала данные сентября 2022 года.

*Что делала:*
 * Произвела анализ данных по транзакциям сети розничных магазинов.

 * Изучила длительность операций, выявила аномалии.

 * Проанализировала результаты маркетинговой компании: оценила влияние акции на выручку, прибыль и средний чек.

 * Расчитала коэффициенты прироста и динамику ключевых метрик.

 * Построила интерактивный дашборд и графики в Google Sheets, визуализирующие результаты анализа


**Формат курсовой** - PowerPoint. Было необходимо представить видео с защитой, презентацию, по которой проводилась защита

Используемые инструменты  ***Excel***, Google Sheets, функции Query, сводные таблицы, визуализации (графики, дашборд).
